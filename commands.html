Commands

Music Command

Mysterio BOT have music commands with high sound quality. Some of the music commands are play,pause,resume,skip,247 etc. The bot can also stay for 24/7 in a vc.

Utility Command

Some of the Utility commands are announce, poll, embed etc. and most importantly Mysterio has a afk command.

Moderator Commands

Mysterio contains moderator commands which can be used to kick, ban, add role, create channel, lock down channel, etc. You can easily moderate your server by using Mysterio.

Fun Commands

This bot have also many commands for fun such as meme,wanted,hitler,dog,cat etc

Info Commands

Mysterio BOT have commands which is related to info. You can use these commands to know about a server or a person. Some commands are userinfo,serverinfo,botinfo etc

Giveaway Commands

Bot also provides the feature of giveaway. You can simply start a giveaway in your server by using the command ,gstart.
